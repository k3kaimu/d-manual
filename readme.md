# このテキストの目的

* 初心者にとって、簡単な入門記事となり得る  
* 上級者にとって、すぐに仕様を確認できる  


# おねがい

　書いている人はD言語の達人ではないので、テキストのミスを発見した場合や改善点がある場合には、プルリクエストやtwitterでリプを送っていただけると修正します。また、classやsharedなどには多少疎いので、だれか一緒に書いてください。


# マークダウンの記法
